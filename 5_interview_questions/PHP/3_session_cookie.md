# Difference btw session and cookie

## PHP

### session

- stored in server
- expired by closing browser

### cookie

- stored in a browser (client-side)
- possible to deside expiration date (not be relevant to browser state)

## JavaScript

### sessionStorage

- stored in a browser (client-side)
- expired by closing browser

### localStorage

- stored in a browser (client-side)
- possible to deside expiration date (not be relevant to browser state)
    - but not flexible than PHP cookie
