# multidimentional array

``` php
$array = [
    "like" => [
        "flower" => "nemophila",
        "fish" => "kisu",
    ],
    "hobby" => "プログラミング",
];
```

``` js
$array = [
    { 
        "like" : [
            {
                "flower" : "nemophila",
                "fish" : "kisu",
            }
        ],
    },
    {
        "hobby" : "プログラミング"
    },
];
```
